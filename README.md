# noteTaker_express

## Application
noteTaker_express is an Heroku deployed application that allow users to create and store new notes or tasks. Once complete, users can then delete their created tasks.

## Technologies Used

<ul>
<li> Node
<li> Express
<li> Github
<li> Heroku 
</ul>
